Frequently Asked Question

Local storage capacity is 5MB/10MB.Local storage is not session-based, it must be deleted via javascript or manually.
Session storage capacity is 5MB.Session storage is session-based and works per window or tab. This means that data is stored only for the duration of a session, i.e., until the browser (or tab) is closed.

Global Scope: A variable exists inside or outside a block. If a variable is declared outside all functions or curly braces ({}), it exists in the global scope.
Block Scope: A variable when declared inside the if or switch conditions or inside for or while loops, are accessible within that particular condition or loop.let & const are also called as block scope.

JavaScript has a runtime model based on an event loop, which is responsible for executing the code, collecting and processing events, and executing queued sub-tasks.
Event loop works in a cyclic manner, where it continually checks whether or not the call stack is empty. If it is empty, new functions are added from the event queue. If it is not, then the current function call is processed.

There are many ways to get undefined. Such as:
  • Variables that is not initialized will give undefined.
  • Function with no return will give undefined.
  • Parameter that is not passed will be undefined.
  • Property that doesn't exist on an object will give undefined.
